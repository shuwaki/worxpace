v2.6.58 ("Solid") Stable

Shortcut panel has been retired. It has been empty since v2.6.13

What's New

  • Finally retired shortcuts panel and status label in POS window
  • Restored the ability to switch price to wholesale in POS window
  • Altered store transactions page to load transactions for the day only

Bug fixes

  • Fixed SIV print out showing 0 as id number
  • Fixed siv print out showing wrong address and phone

Upgrade Notes

  • This update is not compatible with versions < 2.6.50.

v2.6.55 ("Solid") Stable

Short summary: Restored more features that have been broken since 2.6.12.

Shuwaki Worxpace v2.6.55 focuses on stability improvements, while restoring broken functionality.

What's New

  • Included Automatic update support
  • Enabled viewing and printing of inventory by shelf.

Bug fixes

  • Fixed Inventory history not loading.
  • Resolved app crashing when login attempt throws
  • Fixed some summary figures in admin dashboard not showing correct totals
  • Fixed cashier summary not showing correct totals.

Upgrade Notes

  • This update is not compatible with versions < 2.6.50.
  • Henceforth, app will attempt to check for update whenever it starts, you may see prompts.
