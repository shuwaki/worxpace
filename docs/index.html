Overview

Shuwaki Worxpace® is a business management SaaS focusing on point-of-sale, inventory, and reporting. This page gives a short introduction and sample configuration to get started locally.

Getting Started

  1. Download the latest release from the releases page.
  2. Unpack the release to your server or local directory.
  3. Open the application in a browser. For local installs, open index.html from the project root.

Installation

If you are deploying a simple static build for demonstration, copy the entire project to your webroot. For a production deployment consider configuring an HTTPS-enabled web server and backing the application with the recommended server-side components.

Quick checklist

  • Ensure css/, js/ and images/ are present.
  • Update file permissions if needed.
  • Point your web server to the project root where index.html resides.

Sample Configuration

Below is a small example of a local configuration snippet that demonstrates expected keys for a minimal setup.

{
  "company": "Your Business Name",
  "mode": "demo",
  "database": {
    "type": "sqlite",
    "path": "data/worxpace.db"
  },
  "features": {
    "pos": true,
    "inventory": true,
    "reports": true
  }
}
