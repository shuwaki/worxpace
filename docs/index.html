Overview

Shuwaki Worxpace® is a combined POS, inventory management, customer management, supplier management and accounting system for pharmacies. It lends alot of development effort to ease of use and comprehensive reporting. This page gives a short introduction and general overview to enable a smooth start.

Getting Started

  1. Contact Shuwaki Ibrahim (+234 803 859 0989) to register a company and get a super user login.
  2. Download the latest release from the releases page.
  3. Install the downloaded app.
  4. Launch the application and enjoy seamless management of your pharmacy.

Installation

The app runs on Microsoft .Net Desktop Runtime. On the releases page you will the .Net version that is required for the app version you intend to use. Download the appropriate .Net version, then download the latest version of Shuwaki Worxpace®, launch it and get working

Quick checklist

  • Ensure you are on a supported windows OS (Win 7 or later, x64).
  • Ensure you have the appropriate .Net version installed.
  • Ensure you are connected to the internet.
